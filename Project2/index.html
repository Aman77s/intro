How we Clone our project in git and posh changes

  1. We Copy link from our project hosted on github and in our terminal we write a command git clone and paste link
  2. Now we can access our project in our code editor
  3. After changing or adding some code file we need to follow these steps
    • First of all we have to run a command git add . (Now our all changes add in git local repo )
    • After that we run a command git commit -m "commit message"
    • Now we can check our process using git status
    • All set we just need to push our changes into github So we run this command git push origin main

How we can publish local project on github

  1. First of All we need to create a repo on Github and copy link
  2. Now we have to add file so we run a command git init
  3. Step 2 . run git add .
  4. Step 3. run git comit -m "Add Some Files
  5. Run git remote add origin link
  6. git remote -v (to verify remote )
  7. Run git branch (to check branch )
  8. Run git branch -M main (to rename branch name )
  9. Run git origin main
